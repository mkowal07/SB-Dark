## SB Admin Dark

This is an adaption of the [SB Admin Bootstrap](https://startbootstrap.com/template/sb-admin) theme for "dark mode". Almost all changes were made in css/styles.css, with some others happening in assets/demos/chart*.js to adjust the included demo charts. Some changes may not necessarily conform to the coding styles that were followed with the original theme.
